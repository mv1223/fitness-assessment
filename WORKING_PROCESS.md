## **🔧 TROUBLESHOOTING**

### **If npm install fails:**
```bash
npm install --legacy-peer-deps --force
```

### **If Android build fails:**
1. Install Android Studio
2. Set ANDROID_HOME environment variable
3. Run `npm run android` again

### **If backend fails:**
```bash
cd backend
npm install
npm start
```
